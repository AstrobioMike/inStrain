inStrain
====================================

A tool for  analysis of co-occuring genome populations from metagenomes that allows
 highly accurate genome comparisons, analysis of coverage, microdiveristy, and linkage
 and sensitive SNP detection with gene localization and synonymous non-synonymous identification

Source code is `available on GitHub
<https://github.com/MrOlm/instrain>`_.

See links to the left for :doc:`installation` and :doc:`quick_start` instructions

Comments and suggestions can be sent to `Matt Olm <mattolm@berkeley.edu>`_ and/or
`Alex Crits-Christoph <crits-christoph@berkeley.edu>`_

Bugs reports and feature requests can be submitted through `GitHub
<https://github.com/MrOlm/instrain/issues>`_.
